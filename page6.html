What is this project about?

This is a collaborative software engineering project where 9 team members work together to build a multi-page website. Each member contributes a unique page while following modern web development practices, version control with Git, and responsive design principles.

How many team members are working on this project?

There are 9 team members plus a team leader, making a total of 10 people involved in the project. Each member is responsible for creating and maintaining one specific page of the website.

What technologies are used in this project?

The project uses modern web technologies including:

  • HTML5 for structure
  • Tailwind CSS for styling
  • JavaScript for interactivity
  • Git for version control
  • GitHub for collaboration
  • Responsive design principles

Is this website responsive?

Yes, all pages are fully responsive and work on:

  • Desktop computers
  • Tablets
  • Mobile phones
  • Different screen sizes and orientations

We use Tailwind CSS's responsive utilities to ensure optimal viewing experience across all devices.

Can I use this project code for learning?

Absolutely! This project is educational in nature and can be used as a learning resource. You can:

  • Study the code structure
  • Learn about Git collaboration
  • Understand responsive design patterns
  • See practical Tailwind CSS usage

Please give credit if you use significant portions of the code.

How is the team organized?

The team follows a structured organization:

  • 1 Team Leader (Charles) - Overall coordination
  • 9 Team Members - Each responsible for one page
  • Regular meetings and standups
  • Clear division of responsibilities
  • Version control with feature branches

Visit our Team Page to learn more about each member.
